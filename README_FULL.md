## Why This Fork Exists

The fork focuses on maintenance and practical fixes for teams using JSON-to-Dart generation in active projects.

## What Is Different

- Updated maintenance path and release ownership.
- Freezed output improvements and formatting fixes.
- Compatibility updates for modern VS Code/TypeScript setups.
- Reliability improvements for conversion commands and flows.

## Feature Overview

- Convert JSON from clipboard.
- Convert JSON from selected editor text.
- Generate classes for Freezed and Json Serializable.
- Respect null-safety conventions.
- Trigger generation from command palette and context menu.

## Usage

1. Copy or select valid JSON.
2. Open Command Palette.
3. Run a JSON to Dart command.
4. Provide class name and output preferences.
